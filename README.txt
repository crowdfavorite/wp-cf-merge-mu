## CF Merge Network Blog Content

The CF Merge Network Blog Content plugin provides an easy way to merge content from one blog to another in the WordPress admin.  Buttons are provided for merging Posts, Pages and Attachments.

## Usage

Click on the CF Merge Content link under the Super Admin section of the WP Admin Navigation.  Then select the blog to import content, and click the button for the desired content import.

## Notes

Always backup the database before running any kind of database altering script like this one.
